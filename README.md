# Inventory Management System

A simple **Inventory Management System** built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic user authentication pages and a dashboard layout for managing inventory-related data.

## Features

- User Login
- User Registration
- Dashboard Interface
- Clean and simple UI
- Frontend-only implementation

## Technologies Used

- HTML
- CSS
- JavaScript

## Getting Started

Follow these steps to run the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/barath-roshan/inventory_management_system.git
   ```

2. Navigate to the project directory:
   ```bash
   cd inventory_management_system
   ```

3. Open `index.html` in your browser.
